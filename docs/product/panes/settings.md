# Settings Pane

Owner doc for visible settings behavior.

## Sections

- Appearance: theme, scale, font family, and font size.
- Editor: tab width, autosave, line endings, and reduced-feature thresholds.
- Workspace: restore behavior and recent-file retention.
- Browser: external-open policy, permissions, and downloads directory.

## Rules

- Settings changes apply immediately when safe.
- Risky changes require a confirmation notice.
- Settings are persisted as readable TOML.
