# Markdown Preview Pane

Owner doc for visible Markdown preview behavior.

## Behavior

- Preview follows a selected editor buffer.
- Headings, tables, links, code blocks, and lists render as structured content.
- Source cursor can sync to the nearest preview block.
- Preview scroll can follow the editor or be manual.

## Links

- Local file links open through the workspace.
- Web links open in a browser pane or system browser by policy.
- Unsafe or malformed links show a pane-local notice.
