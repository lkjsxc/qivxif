- pane

- media
- search view
- `kjxlkj` board
- settings
- system

## Behavior

- Users create and open nodes from the workspace.
- Node metadata is editable through operations.
- Unknown node kinds are rejected until extension docs exist.
- Tombstoned nodes stay inspectable through history.
