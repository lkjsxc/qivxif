# Edges

## Purpose

Edges are visible relationships between nodes.

## Kinds

- contains
- references
- authored by
- published as
- draft of
- replies to
- mentions
- tagged with
- follows
- mutes

- reposts
- bookmarks
- reacts to

- pane hosts
- layout splits
- layout stacks
- cache pins
- topic of
- derived from

## Behavior

- Edges can be created from graph views and boards.
- Forward and reverse traversal are equally supported.
- Deleted edges become tombstones before compaction eligibility.
