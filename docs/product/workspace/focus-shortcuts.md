# Focus And Shortcuts

Owner doc for focus behavior.

## Focus

- Exactly one pane is focused.
- Editor panes own text input and IME while focused.
- Browser panes take focus only on click or explicit command.
- Explorer keyboard navigation does not depend on pointer hover.

## Movement

- Focus next tile uses Ctrl or Cmd Alt Arrow.
- Move pane uses Ctrl or Cmd Shift Alt Arrow.
- Tab movement stays inside the active tab stack.
- Modal surfaces return focus to the invoking pane.
