# Profile Surfaces

## Surfaces

- Profile tab.
- Edit profile tab or pane-local modal.
- Profile card in Graph Map inspector.
- Profile search.
- Admin profile management surface.
- Public profile route when published.

## Profile Tab

The tab shows avatar, display name, handle, bio, links, visibility, relationship
counts, authored public nodes, posts, and media.

## Profile Search

Search matches display name, handle, aliases, and public metadata visible to the
viewer. Hidden profile fields do not leak through search snippets.

## Admin Surface

Admins can inspect roles, state, audit summaries, and profile node id. Admins do
not see password hashes or token secrets.
