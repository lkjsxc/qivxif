# PvPvE Events

## Scope

This file owns player-facing PvPvE event categories and risk rules.

## Event Types

- Rift zones.
- Dungeons.
- World bosses.
- Extraction resource events.
- Frontier caravans.

## Rules

- Protected zones disable PvP.
- Frontier events permit contested play.
- Rewards should justify risk without forcing every player into combat.
