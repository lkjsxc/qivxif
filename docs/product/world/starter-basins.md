# Starter Basins

## Scope

This file owns starter basin purpose, selection requirements, and travel style.

## Purpose

Starter basins reduce spawn crowding and give new players space to learn.

## Rules

- A basin must have safe footing.
- A basin must have basic resources.
- Selection uses bounded attempts and explicit failure.
- Moving from basin to frontier is a deliberate travel event.

## Teleport Style

Travel uses countdowns, safe-destination checks, success messages, and failure
messages.
