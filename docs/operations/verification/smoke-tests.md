# Smoke Tests

Owner doc for app smoke checks.

## Shell Smoke

- Launch the app in a headless or virtual display environment.
- Create the default workspace.
- Render at least one nonblank frame.
- Exit cleanly.

## Workflow Smoke

- Open a temporary text file.
- Edit and save it.
- Split an editor tile.
- Restore layout from workspace state.
- Render Markdown preview for a fixture.
