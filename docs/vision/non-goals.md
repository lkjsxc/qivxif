# Non-Goals

## Retired

- Local GUI shell as the primary product.
- File tree as the primary data structure.
- Durable state hidden in opaque layout blobs.
- Unreliable datagrams for durable writes.
- External hosted services on the core path.

## Deferred

- Plugin marketplace.
- Black-box social ranking.
- Background sync before foreground sync works.
- Static blog export before dynamic public routes work.
