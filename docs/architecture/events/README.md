# Events

This directory owns durable event contracts.

## Contents

- [event-envelope.md](event-envelope.md): accepted event envelope fields.
- [event-graph.md](event-graph.md): causal and semantic event relations.
- [edge-events.md](edge-events.md): edge creation, tombstone, and relation events.
- [tree-projection.md](tree-projection.md): tree views derived from relations.
- [idempotency.md](idempotency.md): duplicate upload and replay rules.

## Rule

Events are durable facts. Nodes, edges, tile layouts, feeds, text projections,
and Graph Map views are projections of accepted events.
