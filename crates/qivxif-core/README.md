# qivxif-core

Shared identifiers, timestamps, visibility, auth capability types, and core errors.

## Source

- [src/lib.rs](src/lib.rs): crate root exports.
- [src/ids.rs](src/ids.rs): typed node, edge, event, and actor IDs.
- [src/auth.rs](src/auth.rs): capability and auth action types.
- [src/visibility.rs](src/visibility.rs): visibility enums for graph records.
