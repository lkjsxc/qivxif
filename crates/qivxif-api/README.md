# qivxif-api

Axum route handlers, request and response DTOs, setup, sync, and social endpoints.

## Source

- [src/lib.rs](src/lib.rs): router assembly.
- [src/routes.rs](src/routes.rs): route registration.
- [src/setup.rs](src/setup.rs): first-run owner setup API.
- [src/sync.rs](src/sync.rs): push and pull handlers.
- [src/social.rs](src/social.rs): posts, follows, moderation routes.
